# Chat on WebSocket

Chat on WebSocket is a chat built on websockets and rest API

## Installation

```bash
git clone https://github.com/loveyousomuch554/WebSocket-chat.git
```

## Usage

```bash
npm install
```
than
```bash 
npm run start
```
